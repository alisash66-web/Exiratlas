# ETA Social Network (Instagram-like Platform)

## Product Vision
Build a modern industrial + business social network inspired by Instagram Reels, optimized for:
- Industrial suppliers
- B2B trading
- OEM manufacturers
- Engineering companies

## Core Idea
A platform where companies can:
- Post short videos (Reels-style)
- Showcase industrial equipment
- Share RFQs and offers
- Connect buyers and suppliers

---

## MVP Features

### 1. Video Feed (Reels Style)
- Vertical scroll feed
- Auto-play videos
- Like / Comment / Save

### 2. Company Profiles
- Verified companies
- Industry tags (Oil & Gas, Petrochemical, Steel)
- Product catalog

### 3. Marketplace Layer
- Post RFQ (Request for Quotation)
- Post Offers
- Matchmaking system

### 4. Messaging System
- Direct chat between companies
- RFQ negotiation threads

### 5. Search Engine
- Search suppliers
- Search products
- Filter by country and certification

---

## Tech Stack (Recommended)

### Frontend
- Next.js (React)
- TailwindCSS
- Framer Motion

### Backend
- Node.js / NestJS
- PostgreSQL
- Redis (feed caching)

### Media
- S3 compatible storage
- CDN for video streaming

### AI Layer (ETA OS Integration)
- Supplier matching AI
- RFQ parsing agent
- Jarvis recommendation engine
