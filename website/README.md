# ETA Website System (Exir Tejarat Atlas)

## Overview
This is the official web system for Exir Tejarat Atlas (ETA), designed as an AI-powered industrial trading platform.

The website is not just a company page — it is a **lead generation + RFQ + vendor + AI assistant platform**.

---

## Core Modules

### 1. Home Page
- Company introduction
- Core industries
- Global sourcing capability
- CTA: Request RFQ / Become Vendor

### 2. RFQ Portal
- Submit technical inquiry
- Upload datasheets
- Auto-generate RFQ ticket

### 3. Vendor Registration
- Supplier onboarding form
- Company verification fields
- Category selection (Pumps, Valves, Compressors, etc.)

### 4. Product & Industry Pages
- Oil & Gas
- Petrochemical
- Steel Industry
- Power Plants

### 5. AI Assistant (Jarvis)
- Chat-based supplier assistant
- RFQ generator
- Technical Q&A

---

## Recommended Tech Stack

### Frontend
- Next.js (React)
- Tailwind CSS
- TypeScript

### Backend
- Node.js API routes OR Python FastAPI

### Database
- PostgreSQL

### AI Layer
- OpenAI / Claude API
- Internal Agent Router
